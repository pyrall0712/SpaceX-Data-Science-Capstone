# SpaceX Falcon 9 First Stage Landing Prediction

## Project Background
SpaceX offers Falcon 9 launches for $62 million, while competitors cost over $165 million. This price gap is due to SpaceX's ability to reuse the first stage roket.

## Objectives
- Predict first-stage landing success using Machine Learning.
- Analyze factors affecting landing success.
- Create interactive dashboards for data-driven insights.

## Methodology
1. **Data Collection**: Used SpaceX API and Web Scraping from Wikipedia.
2. **EDA**: Performed analysis using SQL and Python (Matplotlib/Seaborn).
3. **Interactive Visuals**: Built maps with Folium and dashboards with Plotly Dash.
4. **Predictive Analysis**: Trained models including Logistic Regression, SVM, KNN, and Decision Tree.

## Key Results
- The Decision Tree model achieved the best training accuracy (~90.5%).
- Landing success rates have significantly improved since 2013.
